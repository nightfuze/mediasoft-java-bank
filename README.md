# Практическое задание 1

### 1. Создайте класс BankAccount:

#### Поля класса:
- имя владельца (строка);
- баланс (целочисленный);
- дата открытия (LocalDateTime);
- поле-флажок заблокирован ли счет (boolean).

### 2. Добавьте один конструктор, который принимает только один параметр - имя владельца.
> [!NOTE]
> Не забудьте инициализировать остальные поля начальными значениями.

### 3. Добавьте методы:

- deposit (amount) – пополнение счета;
- withdraw (amount) – снятие денег;
- transfer (otherAccount, amount) – перевод денег на другой счет, т.е. с текущего счета снять amount и зачислить amount на счет otherAccount).
> [!NOTE]
> Каждый метод должен возвращать значение типа boolean - успешно или неуспешно была проведена операция.

### 4. Проверьте реализованные методы
> [!NOTE]
> Помните, что нельзя перевести и снимать сумму бОльшую, чем есть на счете (поле "баланс").

### Доп.задания *
> [!NOTE]
> Следующие задания делать не обязательно - они сложнее, чем предыдущие, но помогут закрепить знания)

1. Реализуйте метод toString класса Object для вывода в консоль информацию о счете.

2. Добавьте в класс поле number - номер счета в формате XXXXXXXX, где X - цифра от 0 до 9.
Во время создания объекта сгенерируйте его (см. класс Random).

3. Реализуйте метод equals (можно заодно метод hashcode) для сравнения счетов.
